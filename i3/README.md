# i3

## installation

```bash
yay -S i3-gaps
yay -S i3lock
yay -S i3status
```

py3status module dependecies
```bash
yay -S py3status
yay -S python-tzlocal
yay -S python-pytz
yay -S python-pydbus
yay -S python-dbus
pip3 install pygi
```

## configuration

This folder contains the config file for `i3wm` and `py3status` statusbar.

path to file:
- `~/.config/i3/config`
- `~/.config/i3/i3status.conf`
